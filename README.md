## Instructions for running the model
<ol>
<li>Go to http://www.neos-server.org/neos/solvers/lp:CPLEX/AMPL.html</li>
<li>Submit the .mod file as the model file, the .dat file as the data file, and the .commands file as the commands file</li>
<li>Check the “Short Priority” box</li>
<li>Enter your email and prepare for spam from IBM (feel free to enter my HMC email)</li>
<li>Sit on the next page until the solver finishes (~30 seconds)</li>
</ol>
